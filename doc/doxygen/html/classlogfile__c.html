Definition at line 73 of file general-log.inc.

$oops_i_threw_an_exception = false

flag true if the logger throws.

history teaches that errors thrown by the logger function are often due to a lack of free space on the filesystem or corruption on the filesystem. such situations often result in multiple error catching blocks attempting to log their errors - but the logger then tries to throw because it is not able to function. This infinite loop is caught through the use of the flag here.

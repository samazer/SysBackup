Detailed Description

SSH I/O support

This class extends the Base VFS I/O class and overrides the methods that access the filesystem. The new methods use the remote file I/O class to perform the desired operations on the remote system through SSH to provide the remote backupset VFS logic.

_vfs_file_exists ( $  fn)

Check if a file exists

This is one of the basic operations that needs to be done on a VFS in order to get the previously written hashlist. Each VFS which extends this class will have to override this method to provide this functionality on the target filesystem.

For convenience sake the base class implements the functionality required for a simple disk-based filesystem.

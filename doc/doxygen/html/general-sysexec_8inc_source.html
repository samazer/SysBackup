00036  * <p>http://www.php.net/manual/en/function.proc-open.php

00039  * process to be executed: stdin, stdout and stderr. By default the
00040  * current working directory is used for the process and the
00041  * environment of the current process is used for the new

00057  *

00060  *

00067  *

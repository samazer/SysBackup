whoami ( )

whoami()

determines the effective uid() at the time of the function call. Note that this may be different from the effective UID at the time of the object instantiation. Converts the UID into a user name and returns the user name. At the time of this writing this information is intended to be used in error messages to facilitate debugging of system scripts.

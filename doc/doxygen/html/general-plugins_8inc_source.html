00010  *
00011  * <ul>

00014  *     particular directory and load them into an appropriate list
00015  *     structure.
00016  *
00017  * <li>Load any configuration files found. Each configuration file is
00018  *     effectively an event handler that, when called, executes one of
00019  *     the plugins to perform a desired function. The plugin architecture
00020  *     will load all the configuration files, assign the various tasks to
00021  *     the desired events and sort them into the desired order.
00022  *
00023  * <li>Trigger the various events in the desired order, instantiating the

00025  *     the task.
00026  *

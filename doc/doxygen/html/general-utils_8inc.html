dir_fn() concatenate a directory and a file name

Add a filename to a directory name separated by a forward slash. Returns the concatenated string with double-forward-slash characters reduced to single forward slashes.

Parameters:
string$dira folder path with or without a trailing slash
string$fna file name with or without a leading slash
Returns:
string the dir and fn concatenated with extra slashes removed

echo dir_fn( "/my/folder/", "/myfile" ), "\n"; echo dir_fn( "/my/folder", "myfile" ), "\n"; echo dir_fn( "//my//folder//", "//myfile" ), "\n"; echo dir_fn( "", "myfile" ), "\n"; echo dir_fn( "myfile", "" ), "\n"; echo dir_fn( "folder", "file" ), "\n";

// expects: // /my/folder/myfile // /my/folder/myfile // /my/folder/myfile // myfile // myfile // folder/file

job_cleanup ( $  configuration)

Job Cleanup

This method is called for EACH JOB after all the jobs have been exec'd and before the app_after_events() method is called. This is a good time to perform any cleanup operations that might be needed before the plugin is unloaded.

Job Prepare

This method is called for EACH JOB just after the app_before_events() method is called (ie: just before any events are triggered, before any jobs start. This is a good time to emit any error messages that might be generated during job execution.)

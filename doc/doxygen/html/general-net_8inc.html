ifconfig( device )

Returns an array containing the data returned by a call to ifconfig. Returns information for all devices if no device is specified, returns only information for the specified device or false if the device is not listed in /proc/net/dev.

This method returns information that was cached following a call to _ifconfig();

network_device_stats()

Returns the contents of /proc/net/dev as an array. If a specific device is requested then only the metrics for that device are returned.

The result of microtime(true) is added at the end of each metric array as a time stamp for the data. See the detailed comment in the code for an example of the layout of the array.
